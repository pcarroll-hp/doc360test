<h2 id="_additional_installation_requirements">Additional installation requirements</h2>
<div class="sectionbody">
<div class="paragraph">
<p>HP Sure Click Enterprise installation also requires the following:</p>
</div>
<div class="ulist">
<ul>
<li>
<p>A valid Sure Click Enterprise license, provided by your HP Sales or Customer Support representative.</p>
</li>
<li>
<p>An HP cloud-hosted instance of Wolf Security Controller, or a customer on-premise Wolf Security Controller.</p>
</li>
<li>
<p>Trusted Platform Module (TPM) is recommended.</p>
</li>
<li>
<p>Local administrator privileges (if installing on specific machines for evaluation).</p>
</li>
<li>
<p>If you are using <code>msiexec</code> to install Sure Click Enterprise remotely, ensure that the <code>SERVERURL</code> setting is included, otherwise installation fails.</p>
</li>
</ul>
</div>
<div class="admonitionblock note">
<table>
<tr>
<td class="icon">
<div class="title">Note</div>
</td>
<td class="content">
<div class="paragraph">
<p>Do not install Sure Click Enterprise software from a removable drive, such as a USB drive. Removable drives are not trusted by default and, when the initialization stage occurs, the installer fails because it can no longer read the data on the removable drive.</p>
</div>
</td>
</tr>
</table>

<div class="sect3">
<h4 id="_windows_hypervisor_platform_whpmicrosoft_hyper_v">Windows Hypervisor Platform (WHP)/Microsoft Hyper-V</h4>
<div class="ulist">
<ul>
<li>
<p>Support for endpoints running Windows Hypervisor Platform (WHP)/Microsoft Hyper-V and Virtualization-based Security (VBS) with the following configuration:</p>
<div class="ulist">
<ul>
<li>
<p>Windows Hypervisor Platform (WHP) (on Windows 10, version 21H2 - Build 19044 and above).</p>
</li>
<li>
<p>Windows 64-bit with Virtualization-based Security (VBS) enabled.</p>
</li>
<li>
<p>UEFI Secure Boot enabled.</p>
</li>
<li>
<p>The Fast Startup power option in Windows Operating System must be disabled.</p>
</li>
<li>
<p>Intel vPro 4th generation Core (i3/i5/i7/i9) and newer, Xeon, Intel Core Ultra 3, 5, 7, 9, AMD Ryzen, or Qualcomm Snapdragon X, X1, X Plus, X Elite series processors.</p>
</li>
</ul>
</div>
</li>
<li>
<p>Support for non-vPro Intel chipsets.</p>
</li>
<li>
<p>Hibernation/S4 capabilities are disabled and hidden on the host.</p>
</li>
<li>
<p>Running HP Sure Click Enterprise without VMCS Shadowing will result in performance degradations compared to Intel vPro systems. However, HP has taken steps to mitigate performance differentials to all extents possible.</p>
</li>
<li>
<p>Microsoft Hyper-V running on Windows Server 2022. For more information on Hyper-V support in HP Sure Click Enterprise, see <a href="https://enterprisesecurity.hp.com/s/article/How-To-Run-Sure-Click-Enterprise-in-a-Hyper-V-VM">How To Run Sure Click Enterprise in a Hyper-V VM</a>.</p>
</li>
</ul>

<h2 id="_limitations">Limitations</h2>
<div class="sectionbody">
<div class="sect2">
<h3 id="_general">General</h3>
<div class="ulist">
<ul>
<li>
<p>On some Windows Operating System versions, the Credential Protection tile in the HP Wolf Security application does not show a warning when the default browser is an unsupported browser.</p>
</li>
<li>
<p>Microsoft Excel 2019 files shared using <code>Send as PDF</code> sends the email with a text file attachment instead of a PDF.</p>
</li>
<li>
<p>Sure Click micro-VMs support screen readers, but some tools might not be supported. For more information see <a href="https://enterprisesecurity.hp.com/s/article/Enabling-Screen-Reader-Support-in-HP-Sure-Click-Enterprise">Enabling Screen Reader Support in HP Sure Click Enterprise</a>.</p>
</li>
<li>
<p>Saving to and opening untrusted documents from the cloud is not supported for Microsoft Office 365.</p>
</li>
<li>
<p>Temporary trust operation does not trust sites that use "guce-advertising.com" redirect capabilities. The redirects used by this advertising network break a lot of web and software workflows. HP is working to resolve this, but it is a workflow introduced by Verizon Media on most of their web properties. See <a href="https://legal.yahoo.com/ie/en/yahoo/privacy/advertising/index.html">How Digital Advertising Works</a> for more information about this advertising network.</p>
</li>
</ul>
</div>
<div class="ulist">
<ul>
<li>
<p>HP Secure Browser does not support downgrading the browser version. This is a limitation of upstream Chromium.</p>
</li>
</ul>
</div>
</div>
<div class="sect2">
<h3 id="_web_browsing_with_microsoft_edge">Web browsing with Microsoft Edge</h3>
<div class="ulist">
<ul>
<li>
<p>Microsoft Edge&#8217;s support for Microsoft Defender Application Guard does not include extensions which use native messaging. This means that HP Wolf Security Extension (WSX) enters an error state because it cannot connect to its native messaging host.</p>
</li>
</ul>
</div>
<div class="paragraph">
<p>This is a Microsoft issue and not one that can be resolved by HP.</p>
</div>
<div class="admonitionblock note">
<table>
<tr>
<td class="icon">
<div class="title">Note</div>
</td>
<td class="content">
<div class="paragraph">
<p>As of April 4, 2024 Microsoft Defender Application Guard is no longer supported and is considered deprecated by Microsoft. See <a href="https://learn.microsoft.com/en-us/deployedge/microsoft-edge-security-windows-defender-application-guard">Microsoft Edge support for Microsoft Defender Application Guard</a> for more information.</p>
</div>
</td>
</tr>
</table>
</div>
</div>
<div class="sect2">
<h3 id="_web_browsing_with_mozilla_firefox">Web browsing with Mozilla Firefox</h3>
<div class="ulist">
<ul>
<li>
<p>If Firefox has not been launched prior to installing Sure Click Enterprise, you must do the following to ensure browser sessions are isolated in a micro-VM:</p>
<div class="olist arabic">
<ol class="arabic">
<li>
<p>Launch Firefox to create a new profile for the user.</p>
</li>
<li>
<p>Close Firefox and restart Sure Click Enterprise.</p>
<div class="paragraph">
<p>You must follow these steps for each new or additional Firefox user. You must also follow the same steps if you create more than one Firefox profile per user.</p>
</div>
</li>
</ol>
</div>
</li>
<li>
<p>An issue occurs where Firefox is configured as a virtualized browser and set as the default handler for untrusted browser (<code>.html</code>) files. When the file is opened, a tab erroneously appears stating that a navigation is blocked.</p>
</li>
</ul>
</div>
<div style="page-break-after: always;"></div>
</div>
<div class="sect2">
<h3 id="_documents">Documents</h3>
<div class="ulist">
<ul>
<li>
<p>Sure Click Enterprise prevents documents from accessing corporate resources or files stored on the desktop or intranet. As a result, if a document opens in isolation and attempts to connect to a database on the intranet or a linked file on the desktop, it will fail and produce an error. To enable this functionality, you must remove Sure Click Enterprise protection from the document.</p>
</li>
<li>
<p>ASX video files and Windows Update Standalone Installer (MSU) files cannot be opened in micro-VMs.</p>
</li>
<li>
<p>Isolation does not support multiple, simultaneous Microsoft Office installations of the same version.</p>
</li>
<li>
<p>Users might receive an error when opening an isolated file with paths that exceed the Operating System limits.</p>
</li>
</ul>
</div>
<div style="page-break-after: always;"></div>
</div>
