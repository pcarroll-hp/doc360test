<h2 id="_understanding_disk_usage_for_hp_sure_click_enterprise_templates">Understanding disk usage for HP Sure Click Enterprise templates</h2>
<div class="sectionbody">
<div class="admonitionblock important">
<table>
<tr>
<td class="icon">
<div class="title">Important</div>
</td>
<td class="content">
<div class="paragraph">
<p>This article was originally published in English. HP uses machine translation to localize content. If you notice any errors, unclear information, or content that seems misleading, <a href="mailto:wps.support@hpwolf.com">open a support ticket</a>.</p>
</div>
</td>
</tr>
</table>
</div>
<hr>
<div class="paragraph">
<p>This article explains how to accurately measure and understand disk space consumed by HP Sure Click Enterprise on a Windows device. It also explains how to interpret the results.</p>
</div>
<div class="paragraph">
<p>HP Sure Click Enterprise uses advanced storage techniques to optimize disk usage. Due to the use of these techniques, standard Windows disk usage tools can significantly over-report how much disk space HP Sure Click Enterprise actually consumes. The command described in this article reports an accurate view of real disk usage.</p>
</div>
<div class="sect2">
<h3 id="_applies_to">Applies To</h3>
<div class="ulist">
<ul>
<li>
<p>HP Sure Click Enterprise version 4.4 Release 15 (4.4.29.1450) or later.</p>
</li>
</ul>
</div>
</div>
<div class="sect2">
<h3 id="_overview">Overview</h3>
<div class="paragraph">
<p>HP Sure Click Enterprise creates secure and isolated micro-VM using templates. These templates are stored on disk and reused where possible to reduce storage consumption.</p>
</div>
<div class="paragraph">
<p>To achieve this, the product uses:</p>
</div>
<div class="ulist">
<ul>
<li>
<p>Hardlinks, to share common files between templates.</p>
</li>
<li>
<p>Copy-on-write (CoW) storage, so disk space is only used when data changes.</p>
</li>
</ul>
</div>
<div class="paragraph">
<p>Most Windows disk utilities do not account for these mechanisms correctly, which can lead to misleading disk usage figures.  The disk usage tool included with HP Sure Click Enterprise understands this behavior and provides an accurate measurement.</p>
</div>
</div>
<div class="sect2">
<h3 id="_prerequisites">Prerequisites</h3>
<div class="paragraph">
<p>Before you begin, ensure that:</p>
</div>
<div class="ulist">
<ul>
<li>
<p>HP Sure Click Enterprise 4.4 Release 15 (4.4.29.1450) or later is installed.</p>
</li>
<li>
<p>You have permissions to open a command prompt on the device.</p>
</li>
</ul>
</div>
</div>
<div class="sect2">
<h3 id="_how_to_run_the_disk_usage_command">How to run the disk usage command</h3>
<div class="olist arabic">
<ol class="arabic">
<li>
<p>Open <strong>Command Prompt</strong>.</p>
</li>
<li>
<p>Change to the HP Sure Click Enterprise installation directory:</p>
<div class="listingblock">
<div class="content">
<pre>cd %brs%</pre>
</div>
</div>
</li>
<li>
<p>Run the disk usage command:</p>
<div class="listingblock">
<div class="content">
<pre>BrManage.exe init-system disk-usage</pre>
</div>
</div>
</li>
</ol>
</div>
</div>
<div class="sect2">
<h3 id="_example_output">Example output</h3>
<div class="listingblock">
<div class="content">
<pre>TCR00000002
Reason for creation: Machine specification changed (no existing usable template)
Categories: NewInstall|NoTemplate
Stage1: Stage1Image0001 state VALID at C:\ProgramData\Bromium\vSentry\4.4.29.1450\Stage1\Stage1Image0001 size: 2.3 GB (2522002169 bytes) of which 0 B is CoW (0 bytes)
Created: 05/02/2026 at 12:29:19, Last used: 1 minute, and 41 seconds ago
Stage2: Stage2Image0001 state VALID at C:\ProgramData\Bromium\vSentry\4.4.29.1450\Stage2\Stage2Image0001 size: 2.6 GB (2772589459 bytes)
Created: 05/02/2026 at 12:39:01, Last used: 1 minute, and 41 seconds ago

Total disk used by templates: 4.9 GB (5294591628 bytes)</pre>
</div>
</div>
</div>
<div class="sect2">
<h3 id="_understanding_the_output">Understanding the output</h3>
<div class="sect3">
<h4 id="_template_identifiers_tcr_entries">Template identifiers (TCR entries)</h4>
<div class="paragraph">
<p>Each entry starting with <code>TCRxxxxxxx</code> represents a template request generated by HP Sure Click Enterprise. Some requests may not result in a template being created on disk, or may be queued and not yet started.</p>
</div>
<div class="paragraph">
<p>For each template, the output shows:</p>
</div>
<div class="ulist">
<ul>
<li>
<p>Why the template was created.</p>
</li>
<li>
<p>When it was created.</p>
</li>
<li>
<p>When it was last used.</p>
</li>
<li>
<p>How much disk space it consumes.</p>
</li>
</ul>
</div>
<div class="paragraph">
<p>Templates may be created during installation, after system changes, after application updates, or when no compatible existing template is available.</p>
</div>
</div>
<div class="sect3">
<h4 id="_template_creation_reason">Template Creation Reason</h4>
<div class="paragraph">
<p>The <strong>Reason for creation</strong> field explains why the template exists and the trigger for its creation. Examples include:</p>
</div>
<div class="ulist">
<ul>
<li>
<p>Initial installation of HP Sure Click Enterprise.</p>
</li>
<li>
<p>Changes to the operating system or device configuration.</p>
</li>
<li>
<p>Changes to installed applications or the configuration of the application.</p>
</li>
<li>
<p>No existing compatible template available.</p>
</li>
</ul>
</div>
</div>
<div class="sect3">
<h4 id="_stage1_and_stage2_images">Stage1 and Stage2 Images</h4>
<div class="paragraph">
<p>Templates include two stages:</p>
</div>
<div class="ulist">
<ul>
<li>
<p><strong>Stage1</strong>
An initial preparation stage that can be shared across multiple templates.</p>
</li>
<li>
<p><strong>Stage2</strong>
The template-specific user optimizations.</p>
</li>
</ul>
</div>
<div class="paragraph">
<p>For each stage, the output shows:</p>
</div>
<div class="ulist">
<ul>
<li>
<p>Disk location.</p>
</li>
<li>
<p>Size on disk.</p>
</li>
<li>
<p>Creation time.</p>
</li>
<li>
<p>Last usage time.</p>
</li>
</ul>
</div>
</div>
<div class="sect3">
<h4 id="_disk_usage_per_template">Disk usage per template</h4>
<div class="paragraph">
<p>The reported size for each stage reflects actual disk usage, accounting for shared data and copy-on-write behavior. This allows you to identify which templates are consuming disk space and why.</p>
</div>
</div>
</div>
<div class="sect2">
<h3 id="_total_disk_usage">Total disk usage</h3>
<div class="paragraph">
<p>At the end of the output, a summary is displayed:</p>
</div>
<div class="listingblock">
<div class="content">
<pre>`Total disk used by templates: 4.9 GB`</pre>
</div>
</div>
<div class="paragraph">
<p>This value represents the <strong>true disk space</strong> consumed by HP Sure Click Enterprise templates on the device. It accounts for shared files and avoids double-counting data referenced by multiple templates.</p>
</div>
</div>
<div class="sect2">
<h3 id="_key_points">Key points</h3>
<div class="ulist">
<ul>
<li>
<p>Standard Windows tools may significantly over-report disk usage for HP Sure Click Enterprise.</p>
</li>
<li>
<p>The <code>disk usage</code> command provides an accurate, product-aware measurement.</p>
</li>
<li>
<p>Each template entry explains why it exists and how much disk space it consumes.</p>
</li>
<li>
<p>The total reported value reflects real disk impact on the device.</p>
</li>
</ul>
